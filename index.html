HTML/CSS Cheat Sheet

Basic Tags

Tag Name Description
<html> HTML Lets the browser know that the document is HTML
<title> Title Sets the title of the web page (Shown in the browser tab)
<head> Head Contains important links to fonts or style sheets
<body> Body Contains the main code for your HTML document
<div> div Defines a division or section of your HTML code
